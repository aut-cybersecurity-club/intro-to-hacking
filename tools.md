- BurpSuite Community Edition - A proxy, allows you to discover vulnerabilities on web applications (APIs, forms, etc)
- sqlmap - An automated SQL Injection vulnerability scanner - we recommend getting experience manually testing first
- jwt_tool - For testing for the common JWT vulnerabilities - More of an advanced topic
